Some of my Projects

RC Games

This is a social gaming website. Users can create an account on this website and play games which are built in JavaScript and post comments about their experience. The backend of the website runs on PHP and MySQL.

Memory Tiles Game

This is a classical Tile Flipping game in which a tile is flipped up and then a second tile with the same face needs to be flipped up. If a different face shows up, both of them are turned back down and player remembers the face of each tile as he keeps playing, so as to match them all in pairs, in least number of moves. Technologies Used: HTML, CSS, JavaScript, and jQuery.

Puzzle-15 Game

Puzzle-15 is a classical game in which user needs to slide tiles and arrange in linear sequence. This web application is identical to real Puzzle-15 game. The tiles are randomly arranged in solvable configuration at each start.

UFVSUS Clubs Page

The UFVSUS Clubs page shows all the student clubs that are currently active in the university. This page has been completely designed and coded in a new format from scratch. Technologies Used: HTML5, CSS3, jQuery, Javascript, Flex-Box Layout

Amazing Life Hacks
